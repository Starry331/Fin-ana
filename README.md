# FinRisk Pro - 金融风险量化分析平台

一个专业的金融风险量化分析工具，利用Beta系数分析投资风险，支持实时股票数据获取和AI预测功能。

## 功能特性

### 🎯 Beta系数风险分析
- 实时计算股票Beta系数
- 滚动Beta趋势分析
- 相对市场波动性评估

### 📊 全面的风险指标
- **波动率**: 年化价格波动程度
- **夏普比率**: 风险调整后收益
- **索提诺比率**: 下行风险调整收益
- **VaR (95%)**: 日风险价值
- **最大回撤**: 历史最大亏损幅度
- **Alpha**: 超额收益率

### 🤖 智能预测系统
- **ARIMA时间序列预测**: 基于统计模型的自我预测
- **LSTM神经网络预测**: 深度学习AI预测
- 30天价格预测与置信区间
- 预测结果可视化对比

### 📈 多股票对比
- 同时比较多只股票的风险指标
- Beta系数对比图表
- 详细指标对比表格

## 技术栈

### 后端
- Flask (Python Web框架)
- yfinance (Yahoo Finance数据接口)
- pandas & numpy (数据处理)
- scikit-learn (机器学习)
- statsmodels (ARIMA时间序列)
- TensorFlow/Keras (LSTM神经网络)

### 前端
- React 18
- Recharts (图表库)
- Lucide React (图标)
- Axios (HTTP客户端)

## 快速开始

### 1. 安装后端依赖

```bash
cd backend
pip install -r requirements.txt
```

### 2. 启动后端服务

```bash
cd backend
python app.py
```

后端将在 http://localhost:5000 运行

### 3. 安装前端依赖

```bash
cd frontend
npm install
```

### 4. 启动前端应用

```bash
cd frontend
npm start
```

前端将在 http://localhost:3000 运行

## 使用说明

1. **搜索股票**: 在搜索框输入股票代码（如 AAPL, GOOGL, TSLA）
2. **查看风险指标**: 在"风险总览"标签页查看Beta系数和各项风险指标
3. **AI预测**: 切换到"AI预测"标签页，选择预测方法后点击"开始预测"
4. **对比分析**: 在"对比分析"标签页添加多只股票进行风险指标对比

## 风险指标说明

| 指标 | 说明 |
|------|------|
| Beta | β>1表示波动大于市场，β<1表示波动小于市场 |
| 波动率 | 年化标准差，反映价格波动程度 |
| 夏普比率 | (收益-无风险利率)/波动率，>1为佳 |
| VaR(95%) | 95%置信度下的最大日亏损 |
| 最大回撤 | 历史最高点到最低点的最大跌幅 |
